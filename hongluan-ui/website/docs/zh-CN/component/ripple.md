# Ripple 水波指令

使元素点击时生成一个水波扩散的效果。

## 基本用法

:::demo-fs

ripple/basic

:::

## 修改颜色

:::demo-fs

ripple/color

:::

## 禁用水波

:::demo-fs

ripple/disabled

:::



## Attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| color     | 颜色           | string | — | currentColor |
| disabled | 禁用状态 | boolean | - | false |
