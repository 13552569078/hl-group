## Attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| title         | 标题                   | string  | -                          | -      |
| width         | 宽度                   | string  | -                          | -      |
| body-height   | 内容区高度                    | string  | -     | -      |
| body-padding   | 内容区padding                    | string  | -     | -      |
| show-close   | 是否显示关闭icon                    | boolean  | -     | true      |


## Slots
| 方法名 | 说明 | 参数 |
| ---- | ---- | ---- |
| -   | 默认内容插槽 | - |
| title   | 自定义标题 | - |
| action   | 自定义操作区域 | - |
